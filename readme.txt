Redirect users automatically based on their country or IP address, with whitelist and admin log viewer.

== Description ==
GeoSmart Redirect Manager lets you redirect users to different websites or pages based on their country or IP address with a whitelist system.

== Installation ==
1. Upload the folder to /wp-content/plugins/
2. Activate the plugin through the Plugins menu in WordPress
3. Configure settings under "Geo Redirect" in the admin panel
